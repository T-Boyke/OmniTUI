# OmniTUI
Ein modulares, interaktives TUI-Administrations- und Netzwerksystem für Linux (RHEL, Debian, Arch) auf Whiptail-Basis. Bietet vollautomatische Gateway- &amp; Client-Konfiguration, Live-DNS-Benchmarks, Google BBR Tuning, High-Speed-Subnetz-Scanner und einen automatisierten Self-Healing Doctor. 100 % modular, DRY &amp; SOC-konform
